列表 Lists

1. 无序列表 ul（项目之间无先后顺序）

  • HTML —— 负责结构
  • CSS —— 负责样式
  • JavaScript —— 负责行为

2. 有序列表 ol（项目有先后顺序）

  1. 打开浏览器
  2. 输入网址
  3. 回车访问

type="A"（大写字母）：

  1. 选项 A
  2. 选项 B

start="5"（从 5 开始计数）：

  1. 第五条
  2. 第六条

reversed（倒序排名，如排行榜）：

  1. 季军
  2. 亚军
  3. 冠军

3. 描述列表 dl（术语—解释 成对出现）

HTML
超文本标记语言，定义网页结构。
CSS
层叠样式表，定义网页外观。
JavaScript
脚本语言，定义网页交互行为。
