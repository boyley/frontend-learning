08 · 定位 position

static / relative / absolute / fixed / sticky 五种定位详解

relative（相对定位）

relative 相对元素自身原位置偏移，原空间仍被占据（虚线框是它本来的位置）。

原始位置占位 我被 top:20px; left:40px 推开了

absolute（绝对定位）

absolute 脱离文档流，相对最近的非 static 祖先（这里父级设了 position:relative）定位。

钉在右上角 绝对居中（top/left 50% + translate -50%）

z-index（层叠顺序）

定位元素重叠时，z-index 越大越靠上。

z-index:1
z-index:2
z-index:3

五种 position 值对比

值是否脱离文档流定位参照物top/left 等是否生效
static（默认）否正常文档流否
relative否（仍占位）自身原位置是
absolute是最近非 static 祖先是
fixed是视口 viewport是
sticky否滚动容器（阈值前后切换）是（作为吸附阈值）
