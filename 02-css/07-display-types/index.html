07 · 显示类型 display

对比 block / inline / inline-block / none 四种基础值的排布差异

display: block（块级）

每个元素独占一行，宽高、上下 margin 都生效。

盒子 A 盒子 B 盒子 C

display: inline（行内）

多个元素排在同一行，忽略 width/height 与上下 margin（盒子被内容撑开）。

盒子 A 盒子 B 盒子 C

display: inline-block（行内块）

既能同行排列，又能像块级一样设置宽高与上下 margin。

盒子 A 盒子 B 盒子 C

display: none（不渲染）

元素被彻底移除，不占据任何空间。点击按钮切换中间盒子的显隐。

盒子 A 盒子 B 盒子 C

none / visibility:hidden / opacity:0 区别

display:none
（不占位） visibility:hidden
（占位·不可点） opacity:0
（占位·可点） 正常显示
作参照

第一个盒子 display:none 完全消失；第二个 visibility:hidden 留下空白占位；第三个 opacity:0 透明但仍能点击（试着点它左侧透明区域会弹窗）。
