currentColor：边框颜色自动跟随文字颜色（橙色）

2. 长度单位对比（盒子宽度）

px：固定 200px（绝对单位）
%：父元素宽度的 50%
vw：视口宽度的 30%（拖动窗口试试）
ch：约 20 个字符宽

3. em 嵌套放大陷阱

父级 20px（基准）
child 1.5em → 30px
grandchild 1.5em → 45px（逐层叠乘，越来越大！）

4. rem 不会逐层放大

父级 20px
我是 1.5rem，恒等于 24px（相对根字号 16px，与父级无关）

提示：em 相对“自身/父级字号”，rem 相对“根元素字号”，故 rem 更可控。
