CSS 计算函数 calc / min / max / clamp

用数学表达式让尺寸"会算"，做出真正响应式的布局与字号

① calc(100% - 40px) ——混合单位运算

容器宽度 = 父级 100% 减去固定 40px。百分比和像素两种单位只有 calc 能混合相减。

我的宽度 = calc(100% - 40px)，缩放窗口我会一直留 40px 余量

② clamp(MIN, 理想值, MAX) ——流式字号

字号 = clamp(1rem, 2.5vw + 0.5rem, 2rem)。拖动浏览器窗口宽度，下面文字会在 16px~32px 之间平滑变化，到边界自动卡住。

我会随窗口宽度平滑缩放的标题

当前窗口宽度：-px ｜ 实际渲染字号：-px（被锁在 16~32 之间）

④ min() 与 max() ——语义反直觉对比

min() 取较小值当"上限"，max() 取较大值当"下限"。缩放窗口观察两条的宽度变化。

width: min(90%, 320px) → 最宽 320px
width: max(50%, 200px) → 最窄 200px
