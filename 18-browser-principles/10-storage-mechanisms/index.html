🗄️ 浏览器存储机制 Demo

分别演示 localStorage / sessionStorage / Cookie / IndexedDB / Cache Storage。打开 DevTools → Application 面板对照观察。

1 · localStorage

同步、持久（除非手动清）、同源共享、~5-10MB、不随请求发送。


    


    
    

      
2 · sessionStorage

      
同步、仅当前标签页会话（关标签页即清）、新开标签页不共享。

      

        
        
      

      

        
        
        
      

      

    


    
    

      
3 · Cookie

      
~4KB、随每次同源请求自动发送、可设 Max-Age / SameSite。（本地 file:// 未加 Secure/HttpOnly）

      

        
        
      

      

        
        
      

      

    


    
    

      
4 · IndexedDB

      
异步、事务型 NoSQL、可存大量结构化数据 / Blob。用原生 API + 游标遍历。

      

        
      

      

        
        
        
      

      

    


    
    

      
5 · Cache Storage（可选）

      
存 Request/Response 对，PWA 离线基石。并显示 storage 配额估算。

      

        
        
        
      

      

    


  


  

    💡 观察建议：F12 → Application 面板可看到每种存储的真实内容。
    试试：localStorage 存一条后刷新/重开浏览器仍在；sessionStorage 存一条后关标签页再开就没了；
    在 Network 面板看请求头里 Cookie 被自动带上；改 document.cookie 后对照 Application → Cookies。
