📱 PWA 是什么

用 Web 技术构建、体验却像原生 App 的应用

PWA（Progressive Web App，渐进式 Web 应用）是"用 Web 平台技术构建、但能提供接近平台原生应用体验"的应用（MDN 定义）。 它同时拥有网页的可链接（URL 直达、无需应用商店）与原生应用的可安装、可离线、能后台运行两类特性。

🔗 可链接

一个 URL 即可访问，搜索引擎可索引，跨平台单一代码库运行。

⬇️ 可安装

凭 manifest.json 安装到桌面/主屏，独立窗口启动，像原生应用。

📴 可离线

凭 Service Worker + Cache API 拦截请求，断网也能打开。
