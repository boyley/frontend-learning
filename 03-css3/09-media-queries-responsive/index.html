媒体查询与响应式 Media Queries

缩放浏览器窗口宽度，观察指示牌文字 / 颜色与卡片列数随断点切换。

👉 拖动窗口边缘改变宽度：手机(1列) → 平板(2列) → 笔记本(3列) → 桌面(4列)

纯 CSS 驱动：媒体查询改写 CSS 变量，无 JavaScript

移动优先

先写小屏基础样式，用 min-width 逐级增强。

断点设计

按内容换行的「逻辑断点」，不绑定具体设备。

弹性布局

配合 grid / flex / 百分比 / rem 自适应。

viewport

必须有 meta viewport 才能正确缩放。

暗色模式

prefers-color-scheme 跟随系统主题。

减少动效

prefers-reduced-motion 关闭过渡。

指针能力

hover / pointer 区分触屏与鼠标。

方向

orientation 判断横竖屏。

本页用到的媒体查询

  • @media (min-width: 600px) → 平板，两列
  • @media (min-width: 900px) → 笔记本，三列
  • @media (min-width: 1200px) → 桌面，四列
  • @media (prefers-color-scheme: light) → 跟随系统切换浅色主题（改系统外观试试）
  • @media (prefers-reduced-motion: reduce) → 关闭所有过渡动画

现代浏览器也支持范围语法：@media (width >= 600px) 等价于 min-width: 600px。
