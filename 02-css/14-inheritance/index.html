14 · 继承与控制（Inheritance）

某些属性会自动从父元素传给子元素；用 inherit / initial / unset / revert 可以精确控制这一行为。

演示 1 · 默认继承：文本属性传，盒模型不传

我是父元素：color / font / line-height / text-align 都设在我身上。
我是子元素，没有写任何文字样式，但红色、斜体、衬线字体、居中、大行高全是继承来的； 而父元素的红色边框、内边距、浅红背景没有传给我（盒模型与背景不继承）。

演示 2 · 四个关键字显式控制继承

父元素 color 为深蓝 #1d3557： inherit → 深蓝（强制继承） initial → 黑色（重置为初始值） unset(color) → 深蓝（可继承属性=inherit） unset(border) → 无边框（不可继承属性=initial）

revert 则回退到浏览器默认样式表的值（本页未单独演示，规则与上面类似但参照 UA 样式）。

演示 3 · currentColor：让其它属性跟随继承来的 color

本盒子只设了 color: #2a9d8f。边框用 currentColor、下面的小圆点 background 也用 currentColor， 它们都自动变成同一种青色。

这行文字用 color:inherit，同样是青色。

属性是否默认继承（对照）

属性是否默认继承类别
color继承文本
font-family / font-size / font-style继承字体
line-height / letter-spacing继承文本排版
text-align / white-space继承文本
visibility / cursor继承交互/可见
margin / padding / border不继承盒模型
width / height不继承盒模型
background不继承背景
position / display / float不继承布局
