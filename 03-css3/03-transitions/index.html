CSS Transitions 过渡演示

Transition 让属性值变化时平滑动画而非瞬间跳变。核心四件套： transition-property / duration / timing-function / delay。

② hover 综合过渡

鼠标悬停盒子，background / transform / box-shadow / border-radius 同时平滑变化（过渡写在常态上，移开也平滑）。

③ 点击展开 / 收起（max-height 技巧）

高度 auto 无法过渡，改用 max-height 从 48px 过渡到 240px。

点我展开 / 收起 ▼
这是面板内容。点击标题切换 .open 类，max-height 在 0.5s 内平滑变化， 配合 overflow:hidden 裁剪内容，营造「抽屉」展开效果。 背景色也同步过渡。
