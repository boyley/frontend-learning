11 · 伪类 pseudo-classes

伪类（单冒号 :）用来匹配元素的特定状态或在文档树中的特定位置。

① 斑马纹列表（:nth-child / :first-child / :hover）

  • 表头行（:first-child 加粗）
  • 第二行
  • 第三行
  • 第四行
  • 第五行
  • 末行（:last-child 虚线分隔）

奇数行用 :nth-child(odd) 着色；把鼠标移到任意行体验 :hover 高亮。

② 结构伪类公式 :nth-child(3n+1)

12345 678910

被选中（紫色）的是第 1、4、7、10 个。公式 an+b：2n=偶数、odd=奇数、3n+1=每 3 个取第 1 个。

③ 表单状态伪类（:focus / :required / :disabled / :checked）

④ 否定伪类 :not()

激活标签 普通标签 A 普通标签 B 普通标签 C

用 span:not(.active) 让所有非激活标签变灰。
