CSS 渐变 Gradients

渐变本质是一张 <image>，用于 background。包含 linear / radial / conic 及其重复版本。

1. 线性渐变 linear-gradient

沿一条直线过渡。角度 0deg 朝上、顺时针增大；也可用 to right 等关键字。相同百分比的色标可制造「硬边条纹」。

45deg 双色
to right 方向
多色标 0/40/100%
硬边四条纹

2. 径向渐变 radial-gradient

从一个圆心向外辐射。可指定 circle/ellipse 形状与 at 圆心位置。

circle 圆形
ellipse 椭圆
at top left 圆心
高光小球

3. 圆锥渐变 conic-gradient

颜色绕圆心旋转过渡，天然适合做色环与饼图。

色环 color wheel
饼图 pie 40/30/30

4. 重复渐变 repeating-*

把色标定义的图案沿方向无限平铺，做条纹、靶心、扇形非常方便。

repeating-linear 斜条纹
repeating-radial 靶心
repeating-conic 扇形

5. 实战：conic-gradient 百分比进度环

用 conic-gradient 画出进度，再用 ::before 挖空中心成环。拖动滑块改变百分比。

65%

进度由 CSS 变量 --p 驱动：conic-gradient(#34d399 calc(var(--p)*1%), #334155 0)

6. 实战：实时调节线性渐变角度

滑块写入 CSS 变量 --ang，直观感受角度方向（0° 朝上、顺时针）。
