CORS demo

先运行 node server.js（API 跑在 http://localhost:4000）。
本页面如果不是从 localhost:4000 打开（比如用 file:// 双击打开，或用别的端口的静态服务器）， 对 4000 的请求就是跨域请求，会触发 CORS 机制。
打开 DevTools → Network，点下面按钮，重点观察非简单请求前面多出来的那一条 OPTIONS 预检。

① 简单请求（Simple Request）

GET + 无自定义头 → 浏览器直接发，Network 里只有 1 条请求。

② 非简单请求（触发预检 Preflight）

POST + Content-Type: application/json + 自定义头 → 浏览器先发 OPTIONS 预检，通过后才发真正的 POST。Network 里会有 2 条（OPTIONS + POST）。
