4. embed / object 简介

<embed> 与 <object> 是更老的嵌入标签：

  • <embed type="application/pdf" src="..."> 可嵌 PDF 等外部资源（自闭合，无回退内容）。
  • <object data="..." type="...">回退内容</object> 类似但支持标签内写回退内容。
  • 现代开发优先用 iframe（嵌网页）、img（图片）、video（视频）。
