08 · Web Storage（localStorage / sessionStorage）

在下面输入笔记并保存，刷新页面后还在；切换主题也会被记住。对比下方 sessionStorage：关掉标签页就没了。

① 主题切换（存 localStorage，持久化）

切换后刷新页面，主题仍保留 —— 来自 localStorage.getItem('theme')。

② 笔记（localStorage，刷新/重开都在）

输入即自动保存；localStorage 关浏览器再开仍在。

③ 草稿（sessionStorage，关标签页即失效）

刷新还在，但关闭标签页后再开就没了 —— 这就是 sessionStorage。
