11 · 多列布局 Multi-column Layout

用一两行 CSS 就能把连续文本流自动分成多列（报纸 / 杂志排版），并控制列间距、分隔线、横跨标题与防止断裂。

① 报纸三列排版（column-count + column-rule + column-span）

下面整段文字用 column-count:3 自动分三列；大标题用 column-span:all 横跨全部列；列之间用 column-rule 画虚线分隔。

CSS 多列布局快报

多列布局（Multi-column Layout）是 CSS 专门为「连续文本流」设计的排版能力。只要给容器加上 column-count 或 column-width，浏览器就会像报纸一样，把里面的内容从上到下、从左到右自动灌入多个等高的列中，开发者无需手动切分内容。

column-count 指定固定的列数，比如三列；column-width 则指定每列的理想宽度，浏览器会根据容器实际宽度尽可能多地放下若干列，因此更具响应式特性。两者也可以用简写 columns 一起写，例如 columns: 200px 3。

列与列之间的空白由 column-gap 控制，默认值通常是 1em。如果希望在间隙中画一条分隔线，可以使用 column-rule，它的写法和 border 完全一致，支持宽度、线型和颜色，例如 2px dashed gray。

多列布局默认会平衡各列的高度（column-fill: balance），让每一列内容尽量一样高，视觉上更整齐。当内容是大段连续文字时，这种自动平衡非常省心，是 flex 和 grid 难以替代的场景。

需要注意的是，多列适合「内容顺序不重要、可以连续流动」的文本，比如文章正文；而表单、卡片网格这类需要精确二维对齐的布局，仍应使用 flex 或 grid。选对工具，排版才会事半功倍。

② break-inside:avoid 防止卡片被列切断

把若干卡片放进三列容器。绿色卡片加了 break-inside:avoid，整张卡片不会被分到两列；橙色卡片没加（bad），内容多时可能从中间被切断到下一列。
