伪元素让我们可以在不增加 HTML 标签的前提下，对元素的局部内容进行精细化样式控制。比如这段文字的首字被放大下沉，而第一整行被加粗显示，全部由 CSS 伪元素完成，没有任何额外的 span 包裹。
