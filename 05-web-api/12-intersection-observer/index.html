12 · IntersectionObserver 交叉观察器

异步监测元素是否进入视口，实现图片懒加载与曝光埋点（无需监听 scroll）

① 图片懒加载（Lazy Load）

下面方块用 data-src 存放「真图」颜色，滚动到可视区才加载并 unobserve。不联网也能看到效果。

⬇️ 向下滚动，观察方块进入视口时才「加载」

真实项目中是把 data-src 指向图片 URL，进入视口时赋值给 img.src 触发真正的网络加载。

② 曝光统计（Impression Tracking）

元素进入视口（threshold=0.5，即露出一半）时高亮并计数，常用于广告/卡片曝光埋点。

已曝光元素数：0
⬇️ 向下滚动，块露出一半即记一次曝光
