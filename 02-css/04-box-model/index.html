📦 CSS 盒模型 Box Model

每个元素都是一个矩形盒子，由内到外分为 content → padding → border → margin 四层。

① 盒模型四层结构

margin（外边距）
border（边框）
padding（内边距）
content（内容）
margin 外边距 border 边框 padding 内边距 content 内容

② content-box vs border-box（红色虚线为 300px 参照标尺）

content-box（标准盒模型）
↑ 300px 参照线
width:200px
实际占位 = 200 + 20×2 + 10×2 = 260px
border-box（替代盒模型）
↑ 300px 参照线
width:200px
实际占位 = 200px（padding/border 向内挤压内容）

③ margin 折叠（相邻外边距合并）

上块 margin-bottom:40px + 下块 margin-top:30px，实际间距 取较大值 40px，而不是相加 70px。

上方块 · margin-bottom: 40px
下方块 · margin-top: 30px
