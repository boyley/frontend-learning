01 · 前端监控体系总览

错误监控 / 性能监控 / 行为监控 —— 三大类各抓一条，直观看到「监控在监控什么」

🐞 错误监控 Error

  • JS 运行时错误
  • 资源加载错误（img/script/link）
  • 接口请求错误（4xx/5xx）
  • 白屏 / 崩溃

⚡ 性能监控 Performance

  • Web Vitals（LCP/FID/CLS）
  • 页面加载耗时（首屏/DOMReady）
  • 资源加载耗时
  • 接口响应耗时

👣 行为监控 Behavior

  • PV / UV
  • 点击 / 交互
  • 路由跳转
  • 页面停留时长
页面加载会自动采一条性能数据；点击页面任意处会采一条行为数据。
