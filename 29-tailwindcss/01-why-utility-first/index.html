① 传统 CSS / BEM（样式在别处）

按需生成

改一次样式要在 CSS 文件里找到对应规则，还得给每个元素想语义化的名字。

② Tailwind 原子类（样式即 class）

按需生成

样式直接写在 class 上，所见即所得，改哪块看哪块，不用起名字、不用切文件。

一句话

原子化 CSS 用一堆单一职责的小工具类（p-6=padding、rounded-xl=圆角）拼出界面， 把「命名 + 写 CSS + 在两个文件间跳」这三件累人的事，压缩成「在 HTML 里选类名」一件事。
