09 · 浮动与清除 float / clear

文字环绕、高度塌陷及三种清除浮动方案

float 的现代用途：图片文字环绕

头像图片

这是 float 在现代布局中保留的主要用途——让文字围绕图片排版。给图片设 float:left 后，它脱离正常流并靠左，后续的文字会自动填充到它的右侧与下方，形成杂志式的环绕效果。整段布局（多列、居中、等高）如今都交给 flex / grid 处理，float 已退居二线，只在"图文环绕"这种场景里仍然好用。继续多写两行让环绕效果更明显：浮动元素会一直影响后面的行内内容，直到内容高度超过它为止。

清除浮动方案对比

方案原理优缺点
clearfix 伪元素::after 生成块级 + clear:both✅ 推荐，无多余标签、无副作用
overflow:hidden（BFC）触发块格式化上下文，自动包含浮动简洁，但可能裁切溢出内容/阴影
空 div clear:both额外元素阻断浮动❌ 增加无语义标签，不推荐
