04 · 2D 变换 Transforms

拖动滑块实时控制 translateX / rotate / scale / skew，直观感受 transform 的每个函数。

translateX（平移）

移

rotate（旋转）

转

scale（缩放）

缩

skewX（倾斜）

斜

transform-origin（旋转中心点）

红圈标出旋转中心。改变 transform-origin 会改变盒子绕哪个点旋转。

关键提示： transform 不脱离文档流（不影响其它元素布局），但会创建层叠上下文并触发 GPU 合成，性能优秀； translate 的百分比相对自身尺寸；多个函数书写时从右到左依次叠加， 例如 transform: rotate(45deg) translateX(50px) 与 translateX(50px) rotate(45deg) 结果不同； inline 行内元素无法应用 transform（需 inline-block / block）。
